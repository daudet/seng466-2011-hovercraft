SEng 466 Homepage

Welcome to the Project Website for SEng 466. This website will be used to document the hovercraft project that will be undertaken by Craig, Deg and David during the spring 2011 semester at UVic. The semester will be broken down into 3 team projects each with a demonstration and a written report. The content on this website will serve as the project reports required for the course.

The content will be mostly technical and will document problems and issues that we have faced to help future classes with similar projects.
